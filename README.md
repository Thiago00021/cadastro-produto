API REST desenvolvida com **Java 21** e **Spring Boot** para gerenciamento de produtos. O projeto implementa um CRUD completo utilizando **Spring Data JPA** e **H2 Database**, permitindo cadastrar, consultar, atualizar e excluir produtos.

Tecnologias utilizadas:
- Java 21
- Spring Boot 
- Spring Web
- Spring Data JPA
- H2 Database
- Maven
- Lombok
- Bean Validation
- Postman

Os testes da API foram realizados utilizando o **Postman**, validando todas as operações CRUD.

#Objetivos do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento Back-end com Spring Boot, incluindo:

- Desenvolvimento de APIs REST
- Arquitetura em camadas
- CRUD
- Spring Data JPA
- Persistência de dados
- Banco de dados H2
- Validação de dados
- Injeção de Dependências
- Testes utilizando Postman
